# Портал анализа конкурентов — Обсерватория (Competitor Observatory)

> **Экран:** `03_obs` — Обсерватория  
> **Роль:** Мониторинг активности конкурентов в реальном времени. Лента событий, таймлайн изменений, отслеживание цен, контента и рекламы.  
> **Навигация:** Sidebar → Overview | Граф связей | **Обсерватория** | Инсайты  
> **Дизайн-токены:** Наследуются из `dashboard_01_main.md` (секция 1)

---

## 1. Концепция экрана

- **Визуальная парадигма:** Command Center — многопанельный мониторинг
- **Ключевой элемент:** Живая лента событий (Activity Feed) + сравнительные графики
- **Подход:** Хронологический. Что происходит сейчас? Что изменилось за последний период?
- **Метафора:** «Наблюдательная башня» — вы видите всё, что делают конкуренты

## 3. Лента событий — Детали

### 3.1. Карточка события (Event Card)

```
┌─────────────────────────────────────────┐
│ 🔴 ЦЕНА   Pool Hall Pro   2 часа назад │  ← Тип + Источник + Время
│ ─────────────────────────────────────── │
│ Снижение цены на вечерние часы          │  ← Заголовок (text-primary, 14px, semibold)
│                                         │
│ Стоимость аренды стола (19:00–23:00)    │  ← Описание (text-secondary, 13px)
│ снижена с 950₽ до 800₽/час (-15.8%)    │
│                                         │
│ ┌─────────────────────────────────┐     │
│ │ Было: 950₽ → Стало: 800₽       │     │  ← Визуальный diff (bg-surface-alt)
│ │ Δ: -150₽ (-15.8%)              │     │
│ └─────────────────────────────────┘     │
│                                         │
│ Источник: 2GIS · Последняя проверка     │  ← Мета (text-muted, 11px)
│ [📊 Анализ влияния] [🔗 Источник] [📌]  │  ← Действия
└─────────────────────────────────────────┘
```

### 3.2. Типы событий

| Тип | Иконка | Цвет бейджа | Примеры |
|---|---|---|---|
| **Цена** | `💰` | `--status-critical` (снижение) / `--status-success` (рост) | Изменение прайса, акции, скидки |
| **Контент** | `📝` | `--accent-cyan` | Новый пост, обновление сайта, новое меню |
| **Реклама** | `📣` | `--topic-potential` (#F97316) | Новая рекламная кампания, таргет |
| **Отзывы** | `⭐` | `--accent-primary` | Новые отзывы, изменение рейтинга |
| **Структурные** | `🏗️` | `--text-secondary` | Ремонт, новые столы, расширение |
| **HR/Персонал** | `👤` | `--text-muted` | Вакансии, смена руководства |
| **Юридические** | `⚖️` | `--status-warning` | Регистрация ИП/ООО, лицензии |

### 3.3. Состояния карточки события

| Состояние | Визуал |
|---|---|
| **Новое (Unread)** | Яркая левая граница `3px solid --status-critical/warning`, фон чуть светлее |
| **Прочитано** | Стандартный фон `--bg-surface`, тонкая граница `--border-subtle` |
| **Закреплено** | Иконка 📌 активна, карточка вверху ленты |
| **Архив** | Приглушённый текст, свёрнутый вид |

## 7. Микроинтеракции

| Элемент | Действие | Эффект |
|---|---|---|
| Event Card (Hover) | Mouse over | `--bg-surface-hover`, `translateY(-1px)`, тень |
| Event Card (Click) | Click | Раскрытие деталей (expand) или переход |
| Tab Filter | Click | Мгновенная фильтрация, fade-анимация карточек |
| Chart Point (Hover) | Mouse over | Tooltip с данными, подсветка связанной карточки в ленте |
| New Event | Появление | `slideDown` + flash-подсветка фона |
| Pin (📌) | Click | Анимация закрепления, карточка всплывает вверх |
| Notification Badge | New event | Bounce-анимация числа |

## 8. Real-time обновления

### 8.1. Механизм
- **WebSocket / SSE** для push-уведомлений о новых событиях
- **Polling fallback:** Каждые 5 минут для не-критичных данных
- **Визуал:** Новые события появляются сверху ленты с анимацией `slideDown`

### 8.2. Индикатор обновления
```
┌─────────────────────────────────┐
│ ● Live · 3 новых события [↑]    │  ← Плавающий бар сверху ленты
└─────────────────────────────────┘
```
- Клик на `[↑]` — скролл к новым событиям

## 9. Инструкции для генерации

1. **Лента:** Виртуальный скролл (react-virtual или intersection observer) для производительности
2. **Графики:** D3.js для line chart и bubble chart, CSS Grid для heatmap
3. **Фильтры:** Instant-фильтрация без перезагрузки (JS filter на клиенте)
4. **Layout:** CSS Grid `1fr 1fr` для split, переключается на стек `< 1024px`
5. **Анимации:** `transition: all 0.2s ease` для hover, `@keyframes slideDown` для новых карточек
6. **Responsive:** При `< 768px` — лента full-width, графики прячутся за табы
