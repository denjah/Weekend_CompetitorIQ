# Портал анализа конкурентов — Инсайты (AI Insights)

> **Экран:** `04_ins` — AI Инсайты  
> **Роль:** AI-генерируемые рекомендации, сравнительные отчёты, прогнозы, стратегические выводы. Диалог с AI-аналитиком.  
> **Навигация:** Sidebar → Overview | Граф связей | Обсерватория | **Инсайты**  
> **Дизайн-токены:** Наследуются из `dashboard_01_main.md` (секция 1)

---

## 1. Концепция экрана

- **Визуальная парадигма:** AI-First Dashboard — чат с аналитиком + структурированные отчёты
- **Ключевой элемент:** Центральный AI Chat + панели с готовыми инсайтами
- **Подход:** Actionable Intelligence — каждый инсайт ведёт к конкретному действию
- **Метафора:** «Стратегический советник» — AI, который знает ваш рынок

## 3. AI Chat — Детали

### 3.1. Сообщение пользователя

```
┌──────────────────────────────────────┐
│                            You 14:32 │
│   Какие главные угрозы от            │
│   Pool Hall Pro в следующем          │
│   квартале?                          │

│                    │ 🟣 DP    │       │  ← Аватар
│                    └──────────┘       │
└──────────────────────────────────────┘
```

- **Фон:** `--bg-surface-alt`
- **Выравнивание:** Справа
- **Аватар:** Круглый, 32px, `--accent-primary` фон

- **Markdown:** Поддержка **bold**, списков, таблиц, code blocks

### 3.3. AI Chat Input

```
┌──────────────────────────────────────────────┐
│ 💬 Спроси AI-аналитика...                    │
│                                              │
│ Suggested:                                   │
│ [Сравни цены с конкурентами]                 │
│ [Прогноз на Q3 2026]                         │
│ [SWOT-анализ Pool Hall Pro]                  │
│                                         [▶] │
└──────────────────────────────────────────────┘
```

- **Input:** Капсульный, multi-line, авто-расширяется
- **Suggested Prompts:** Горизонтальная лента под инпутом (капсульные чипы)
- **Кнопка отправки:** `--accent-gradient`, скруглённый квадрат
- **Focus:** Glow `--accent-primary-glow`
- **Горячие клавиши:** `Enter` = отправить, `Shift+Enter` = новая строка

### 3.4. Контекст чата

```
┌──────────────────────────────────────┐
│ Контекст: [Weekend Бильярд ×]       │  ← Чип с текущим бизнесом
│           [Pool Hall Pro ×]         │  ← Прикреплённые конкуренты
│           [+ Добавить @]            │  ← Расширение контекста
└──────────────────────────────────────┘
```

- Расположен над input
- **@ mention:** Автодополнение конкурентов при вводе `@`
- **Чипы:** Капсульные, `--bg-surface-alt`, крестик для удаления

## 4. Готовые инсайты — Карточки

### 4.1. Типы карточек инсайтов

| Тип | Иконка | Цвет акцента | Описание |
|---|---|---|---|
| **Угроза** | `⚠️` | `--status-critical` | Критические конкурентные угрозы |
| **Возможность** | `🚀` | `--status-success` | Рыночные возможности |
| **Тренд** | `📈` | `--accent-cyan` | Рыночные тренды |
| **Рекомендация** | `💡` | `--accent-primary` | Конкретные действия |
| **Прогноз** | `🔮` | `--topic-direct` (#A855F7) | Прогнозы на основе данных |
| **Сравнение** | `⚖️` | `--status-warning` | Сравнительный анализ |

### 4.2. Структура карточки инсайта

```
┌─────────────────────────────────┐
│ ⚠️ УГРОЗА           Сегодня    │  ← Тип (badge) + Дата
│ ────────────────────────────── │
│ Ценовое давление                │  ← Заголовок (16px, semibold)
│ усиливается                     │
│                                 │
│ Pool Hall Pro и Бильярд-Сити    │  ← Описание (14px, regular)
│ снизили цены на 10-15%.         │
│ Ваша позиция под давлением      │
│ в сегменте «вечерние часы».     │
│                                 │
│ ┌─────────────────────────┐     │
│ │ Влияние: 🔴 Высокое     │     │  ← Метрика влияния
│ │ Уверенность AI: 87%     │     │  ← Confidence score
│ └─────────────────────────┘     │
│                                 │
│ [📊 Подробнее] [💬 Обсудить с AI]│  ← Действия
│ [✓ Принять к сведению]          │
└─────────────────────────────────┘
```

- **Фон:** `--bg-surface`
- **Верхняя граница:** `2px solid` цвет типа (accent карточки)
- **Hover:** `translateY(-2px)`, усиление тени
- **Click «Подробнее»:** Раскрывает расширенный анализ в модальном окне или в чате

### 4.3. Grid инсайтов
- **Desktop:** 3 колонки, `gap: 16px`
- **Tablet:** 2 колонки
- **Mobile:** 1 колонка
- **Сортировка:** По важности (default) / По дате / По типу

## 8. Микроинтеракции

| Элемент | Действие | Эффект |
|---|---|---|
| AI Response | Появление | Typing indicator → Streaming text (посимвольный вывод) |
| Insight Card (Hover) | Mouse over | `translateY(-2px)`, усиление тени, подсветка верхней границы |
| Suggested Prompt | Click | Вставка текста в input + auto-focus |
| Report Download | Click | Progress bar → «Готово ✓» с fade |
| Context Chip | Click `×` | Анимация удаления (shrink + fade) |
| Quick Action | Click | Pulse-эффект → промпт вставляется |
| Chat Scroll | New message | Auto-scroll вниз + smooth |
| @ Mention | Typing `@` | Dropdown с автодополнением конкурентов |

## 10. Инструкции для генерации

1. **AI Chat:** Streaming response (Server-Sent Events), markdown rendering
2. **Инсайт-карточки:** CSS Grid `repeat(auto-fill, minmax(300px, 1fr))`
3. **Отчёты:** Генерация через серверную часть (pptx skill, PDF generator)
4. **Layout:** `display: grid; grid-template-columns: 40% 60%` для desktop
5. **Responsive:** При `< 1024px` — чат full-width, инсайты — отдельный таб
6. **Анимации:** CSS `@keyframes` для typing indicator, `transition` для hover-эффектов
7. **Markdown:** Использовать marked.js или подобную библиотеку для рендеринга ответов AI
