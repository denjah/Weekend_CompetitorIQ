# Портал анализа конкурентов — Граф связей (Competitive Network)

> **Экран:** `02_knot` — Граф связей  
> **Роль:** Визуализация конкурентной экосистемы: связи между конкурентами, поставщиками, площадками, аудиториями. Интерактивный mind-map.  
> **Навигация:** Sidebar → Overview | **Граф связей** | Обсерватория | Инсайты  
> **Дизайн-токены:** Наследуются из `dashboard_01_main.md` (секция 1)

---

## 1. Концепция экрана

- **Визуальная парадигма:** Split-screen — интерактивный холст (65%) + информационная панель (35%)
- **Ключевой элемент:** Force-directed graph (D3.js) — динамический граф связей
- **Подход:** Данные-центричный. Минимум декора, максимум информации
- **Плотность:** Высокая информационная плотность. Панели наложены поверх холста

## 3. Граф связей — Детали

### 3.1. Типы узлов (Nodes)

| Тип узла | Форма | Размер | Цвет обводки | Контент |
|---|---|---|---|---|
| **Вы (Your Business)** | Круг | 80px | `--accent-primary` (glow) | Логотип + имя |
| **Прямой конкурент** | Круг | 60px | `--topic-direct` (#A855F7) | Аватар + рейтинг |

## 5. Состояния экрана

### 5.1. Ничего не выбрано (Default)
- Инфо-панель показывает сводку по всей сети:
  - Общее количество связей
  - Плотность сети
  - Самые сильные конкуренты (топ-3)
  - Последние изменения в связях

### 5.2. Выбран узел (Node Selected)
- Узел подсвечен `--accent-primary` с glow
- Связи от узла подсвечены ярко, остальные затемнены (`opacity: 0.15`)
- Инфо-панель заполнена данными конкурента

### 5.3. Выбрана связь (Edge Selected)
- Связь подсвечена, оба узла активны
- Инфо-панель показывает: тип связи, общие параметры, сравнительный анализ двух сущностей

## 6. Микроинтеракции

| Элемент | Действие | Эффект |
|---|---|---|
| Node (Hover) | Mouse over | Подсветка обводки + tooltip с именем и рейтингом |
| Node (Click) | Click | Выделение + инфо-панель заполняется данными |
| Node (Drag) | Drag | Перемещение узла, пересчёт связей (force simulation) |
| Edge (Hover) | Mouse over | Подсветка линии + оба связанных узла |
| Canvas (Scroll) | Scroll | Zoom in/out (0.5x – 3x) |
| Canvas (Drag) | Drag (на пустом месте) | Панорамирование |
| Node (Double click) | Dbl-click | Переход в полный профиль конкурента |

## 7. Инструкции для генерации

1. **Граф:** D3.js `d3-force` — force-directed layout с collision detection
2. **Canvas:** SVG (до 100 узлов) или Canvas/WebGL (100+ узлов)
3. **Инфо-панель:** Чистый HTML/CSS, данные подгружаются по клику на узел
4. **Тулбар:** Абсолютно позиционированный `position: fixed`, `bottom: 24px`
5. **Анимации:** Физическая симуляция D3 (alpha decay), transitions при выборе
6. **Responsive:** При `< 1024px` — инфо-панель становится bottom sheet (выдвигается снизу)
