# Портал анализа конкурентов — Главный дашборд (Overview)

> **Экран:** `01_main` — Сводная панель  
> **Роль:** Точка входа. Агрегированный обзор всей конкурентной среды, ключевые KPI, тренды, алерты.  
> **Навигация:** Sidebar → Overview | Граф связей | Обсерватория | Инсайты

## 1. Дизайн-система (общая для всех экранов)

### 1.1. Концепция
- **Тема:** Dark Mode, Premium SaaS, Data-Driven Intelligence
- **Настроение:** Профессиональное, аналитическое, технологичное
- **Подход:** High Data Density + чистый воздух между блоками. Модульная сетка. Строгая иерархия через типографику и цвет

## 4. Компоненты Overview

### 4.1. Top Header
- **Поисковая строка:** Капсульная, ширина `400px`, по центру. Иконка лупы слева, плашка `⌘ K` справа
- **Кнопки справа:** «Invite» (контурная капсула), иконка настроек

### 4.2. KPI-карточки (4 штуки в ряд)

Каждая карточка:
```
┌─────────────────────┐
│ 📊 Total Competitors│  ← Иконка + подпись (text-secondary, 12px)
│ 15                  │  ← Значение (IBM Plex Mono, 36px, text-primary)
│ ▲ +3 за месяц       │  ← Дельта (status-success, 14px)
└─────────────────────┘
```

**Набор KPI:**
| KPI | Описание | Иконка |
|---|---|---|
| Конкуренты в мониторинге | Общее количество отслеживаемых | `📊` |
| Средний ценовой индекс | Ваша цена vs рынок | `💰` |
| Угрозы (активные) | Критические изменения конкурентов | `⚠️` |
| Sentiment Score | Средняя оценка репутации | `📈` |

### 4.3. Рыночная позиция (Scatter Plot)
- **Оси:** X — Цена, Y — Качество/Рейтинг
- **Точки:** Каждый конкурент = точка, размер = доля рынка
- **Ваша позиция:** Выделена `--accent-primary` с glow-эффектом
- **Реализация:** D3.js или Chart.js

### 4.4. Ценовой радар (Radar Chart)
- **Оси:** Основные услуги/товары (6–8 осей)
- **Наложение:** Ваш профиль (accent-primary, заливка с opacity) vs топ-3 конкурента
- **Легенда:** Цветные точки + имена

### 4.5. Динамика активности конкурентов (Area Chart)
- **Тип:** Stacked Area Chart
- **Данные:** Посты в соцсетях, обновления сайта, рекламные кампании, акции
- **Период:** Последние 30 дней, с переключателем (7д / 30д / 90д / 1г)
- **Цвета:** Каждая метрика — свой оттенок из палитры акцентов
- **Заливка:** Градиент от цвета линии к прозрачному `--bg-surface`

- **Контент:** Иконка статуса, заголовок проблемы, источник + время, кнопка действия

### 4.7. Таблица конкурентов (Data Table)

| Конкурент | Тип | Рейтинг | Цена (ср.) | Δ Цена | Активность | Угроза |
|---|---|---|---|---|---|---|
| Pool Hall Pro | Прямой | 4.7 ⭐ | 800₽/ч | ↓ -15% | 🔴 Высокая | ⚠️ Критично |

- **Сортировка:** По любому столбцу (клик на заголовок)
- **Фильтры:** Тип конкурента, уровень угрозы
- **Hover:** Строка подсвечивается `--bg-surface-hover`
- **Клик:** Переход в карточку конкурента (Граф связей → детали)

### 4.8. AI Prompt Input (Footer)
- **Расположение:** Нижняя часть контент-зоны, sticky или в конце скролла
- **Форма:** Капсульный инпут во всю ширину
- **Плейсхолдер:** «Задай вопрос AI-аналитику... Используй @ для выбора конкурента»
- **Кнопка:** Справа — иконка отправки с градиентом `--accent-gradient`
- **Focus:** `box-shadow: 0 0 10px var(--accent-primary-glow)`

## 5. Микроинтеракции и динамика

| Элемент | Эффект | CSS/JS |
|---|---|---|
| Карточки | Hover → `--bg-surface-hover` + `translateY(-2px)` | `transition: all 0.2s ease` |
| KPI-значения | Анимация от 0 до значения при загрузке | CSS `@keyframes` или CountUp.js |
| Графики | Плавное появление (draw animation) | D3.js transitions |
| AI Alerts | Новые алерты — fade-in + slide-down | `animation: slideIn 0.3s ease` |
| Строки таблицы | Hover → подсветка строки | `background-color transition` |
| Sidebar пункты | Hover → фон + текст меняют цвет | `transition: all 0.15s ease` |

## 6. Адаптивность

| Breakpoint | Изменения |
|---|---|
| `> 1440px` | Полный layout, 4 KPI в ряд |
| `1024–1440px` | 2 KPI в ряд, графики стекаются |
| `768–1024px` | Sidebar коллапсирует в иконки (60px) |
| `< 768px` | Sidebar скрыт (бургер-меню), 1 колонка |

## 7. Инструкции для генерации

1. **Порядок сборки:** `:root` CSS-токены → Sidebar → Header → KPI Row → Charts → Table → AI Input
2. **CSS:** Grid для основного layout, Flexbox для внутренностей карточек
3. **Графики:** D3.js (предпочтительно) или Chart.js для визуализаций
4. **Шрифты:** Google Fonts — `Inter` (UI) + `IBM Plex Mono` (цифры)
5. **Иконки:** Lucide Icons (outline style, 20px)
6. **Компоненты:** Переиспользуемые классы `.card`, `.badge`, `.status-dot`, `.kpi-card`, `.pill-button`
